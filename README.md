# Multi-Step Form (React + Tailwind)

## Project Description

This project is a multi-step form built with React. It guides users through a sequence of steps to collect personal information, address details, and final confirmation. The form includes validation, step navigation, and conditional access control between steps.

---

## Live Demo

👉 https://andriyParashchuk.github.io/multi-step-form

---

## Tech Stack

* React (with TypeScript)
* Vite
* Tailwind CSS
* PostCSS

---

## How to Run Locally

1. Clone the repository:

```bash
git clone git@github.com:andriyParashchuk/multi-step-form.git
cd multi-step-form
```

2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm run dev
```

4. Open in browser:

```
http://localhost:5173
```
